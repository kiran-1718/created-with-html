💌 Cartinha Animada com Música
Um projeto simples e romântico utilizando HTML, CSS e JavaScript. O código apresenta uma carta interativa com um coração pulsante; ao ser aberta, uma música de fundo começa a tocar automaticamente.

✨ Funcionalidades
Coração Pulsante: Animação suave de "batida" usando CSS Keyframes.

Interatividade: A carta abre ao ser clicada pelo usuário.

Trilha Sonora: Ativação de áudio sincronizada com a abertura da carta.

Design Responsivo: Adaptado para visualização em dispositivos móveis e desktop.

🚀 Tecnologias Utilizadas
HTML5: Estrutura da carta e elementos de áudio.

CSS3: Estilização, posicionamento e animações (pulsar e abrir).

JavaScript: Lógica para disparar o áudio e alternar as classes de animação.

🛠️ Como usar
Clone este repositório ou baixe os arquivos.

Certifique-se de que o arquivo de música (ex: musica.mp3) está na mesma pasta que o index.html.

Abra o arquivo index.html em qualquer navegador moderno.

Clique na carta para ver a mágica acontecer!

📝 Notas
Alguns navegadores podem bloquear o áudio automático por políticas de segurança; por isso, a música é ativada através do clique de abertura da carta.
